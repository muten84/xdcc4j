Welcome to Xdcc4j.

This project offers an extendible REST API and a simple GUI to manage the search and the downloads from a XDCC Bot providing files on an IRC server. The project is modular and there is a simple web GUI that show how you can use the REST API.

Rest Service

The base facade is the XdccDownloaderService interface. All methods exposed in this interface provide high level operations to manage:

  • searching
  • caching
  • start of a file transfer
  • cancel of a file transfer
  • check server connectivity
  • listing of user and channels

WEB GUI

The xdcc-ui module provide a simple web GUI showing how you can use high level interface.
